## 1. 화면 구조 개요

### 1.1 화면 맵

```mermaid
graph TD
    Home[홈 - 주제 입력] --> Loading[생성 중]
    Loading --> Course[커리큘럼]
    Course --> Chapter[챕터 상세]
    Chapter --> Concept[개념]
    Chapter --> Exercise[실습]
    Chapter --> Quiz[퀴즈]
```

### 1.2 주요 화면

| 화면 | MVP | 우선순위 |
|------|-----|---------|
| 홈 (주제 입력) | ✅ | P0 |
| 생성 중 | ✅ | P0 |
| 커리큘럼 | ✅ | P0 |
| 챕터 상세 | ✅ | P1 |
| 개념/실습/퀴즈 뷰어 | ✅ | P1 |
| 히스토리 | ❌ | P2 |

## 3. 컴포넌트

### 3.1 공통

- **Header**: 로고 + 사용자 메뉴
- **Card**: 제목 + 내용 + 액션
- **ProgressBar**: 진행률 표시
- **Button**: Primary/Secondary

### 3.2 페이지별

- **TopicForm**: 주제 입력 + 옵션
- **CourseList**: 챕터 카드 리스트
- **MarkdownViewer**: 개념 정리 렌더링
- **ExerciseViewer**: 실습 문제 표시
- **QuizViewer**: 퀴즈 표시
